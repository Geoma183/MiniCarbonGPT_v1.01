# Data Statement

The repository provides source code, configuration templates, evaluation protocols, and selected public or sample data files. Some materials used in the study cannot be redistributed in full because they originate from third-party databases, publisher-controlled full texts, or educational platforms with uncertain redistribution permissions.

The manuscript should distinguish:

1. Fully public materials: code, configuration templates, metadata, evaluation scripts, and derived aggregate results.
2. Partially public materials: benchmark samples and metadata where full redistribution is restricted.
3. Restricted materials: third-party full-text documents and any source files whose license does not permit redistribution.

Recommended manuscript wording:

```text
The source code, evaluation scripts, configuration templates, and reproducibility materials are publicly available at https://github.com/Geoma183/MiniCarbonGPT_v1.01. Due to licensing restrictions, third-party full-text documents and some benchmark items derived from external educational platforms are not redistributed in full; instead, the repository provides metadata, sample files, processing scripts, and evaluation protocols needed to reproduce the workflow with legally obtained data.
```
